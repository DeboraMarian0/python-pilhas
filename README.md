# Gerenciador de Processos em Python

Este projeto foi desenvolvido em Python com o objetivo de simular um sistema simples de gerenciamento de processos utilizando uma estrutura de dados do tipo **Pilha (Stack)**.

O sistema permite:

- Inserir processos
- Executar processos
- Visualizar a pilha de processos
- Encerrar o programa

O funcionamento segue o conceito:

> **LIFO (Last In, First Out)**  
> O último processo inserido será o primeiro a ser executado/removido.

---

- Python 3
- Biblioteca `collections.deque`
